# Design System Audit

This directory contains the audit specification split into focused files.

## Quick Start

Say **"run audit"** or **"start audit"** to begin.

## Files

| File | Purpose | When to Use |
|------|---------|-------------|
| `patterns.md` | Regex patterns | Finding violations quickly |
| `rules.md` | What to check | Understanding compliance rules |
| `files.md` | File priority | Knowing what to audit first |
| `output.md` | Report format | Generating audit reports |
| `protocol.md` | Execution steps | Running the full audit |
