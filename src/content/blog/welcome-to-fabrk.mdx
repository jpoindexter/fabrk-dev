# Welcome to Fabrk

We're excited to introduce **Fabrk** - a premium SaaS boilerplate built for developers who want to ship fast without sacrificing quality.

## Why We Built Fabrk

After building multiple SaaS applications, we noticed the same pattern: spending weeks setting up authentication, payments, multi-tenancy, and all the other "boring but necessary" features before even starting on the actual product.

We created Fabrk to solve this problem.

## What Makes Fabrk Different?

### 1. **The Anti-Bloat Philosophy**

While competitors ship 800+ files, Fabrk maintains just **156 essential files**. Every line of code serves a purpose. No bloat, no complexity, just what you need.

### 2. **Modern Stack**

- **Next.js 15** with App Router
- **React 19** with Server Components
- **NextAuth v5** for authentication
- **Stripe** for payments
- **TypeScript** in strict mode
- **Prisma ORM** with PostgreSQL

### 3. **Production-Ready Features**

- ✅ Multi-tenancy with RBAC
- ✅ Webhooks system (22 events)
- ✅ API keys with permissions
- ✅ 2FA/MFA authentication
- ✅ Real-time with Pusher
- ✅ Admin dashboard
- ✅ Audit logging

### 4. **Developer Experience**

- **100 production-ready components**
- **64% test coverage** (931+ tests)
- **95% Storybook coverage**
- Comprehensive documentation
- Clean, maintainable code
