## Response Templates

### For Product Hunt Comments

**"Congrats on the launch!"**
```
Thank you so much! Really appreciate the support. Let me know if you have any questions about Fabrk!
```

**"How is this different from [X]?"**
```
Great question! The main differences are:
- More components (87 vs ~40-60)
- 8 ready-to-use page templates
- 20 themes out of the box

Happy to go deeper on any specific comparison!
```

**"What's the refund policy?"**
```
30-day money-back guarantee, no questions asked. If it's not the right fit, just email me and I'll process the refund immediately.
```

**"Is there a demo?"**
```
Yes! You can see everything in action here: [DEMO_URL]

Browse all components, templates, and the landing page live.
```

**"Why $299?"**
```
I priced it based on the time it saves. Building all of this from scratch takes 73+ hours minimum.

At a modest $50/hour, that's $3,650+ of development time.

$299 for unlimited projects felt fair for both sides.
```

**"Looks great!"**

Thanks! Let me know if you try it out - always looking for feedback.

**"Can it do [X]?"**

[Answer honestly - if yes, explain how. If no, acknowledge it's a good idea for future versions]

No need to switch if [competitor] works for you!

Fabrk might be interesting if you want:

### For Email Replies

**"Is this for beginners?"**
```
Fabrk works best if you have some Next.js/React experience.

The components and templates are ready to use, but you'll be customizing TypeScript code and connecting to your own APIs.

If you're comfortable with React basics, you'll be fine. The documentation covers everything step by step.
```

**"Can I use this for client projects?"**
```
Absolutely! The license allows unlimited projects - personal, commercial, client work, anything.

The only restriction is you can't resell the boilerplate itself.
```

## If Things Go Wrong

### Website Down
1. Check hosting provider status
2. Restart if needed
3. Post update on Twitter
4. Apologize and provide ETA

### Checkout Issues
1. Test checkout yourself
2. Check Stripe dashboard
3. Manually process any failed orders
4. Email affected customers

### Negative Feedback
1. Don't get defensive
2. Acknowledge the concern
3. Explain your reasoning
4. Thank them for feedback
5. Consider if change is needed

### Low Engagement
1. Don't panic
2. Reach out to supporters personally
3. Share in more communities
4. Focus on quality over quantity
5. Remember: slow starts can still succeed
