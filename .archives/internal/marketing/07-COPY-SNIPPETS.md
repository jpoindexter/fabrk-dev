## FAQ Copy

### What is Fabrk?
```
Fabrk is a production-ready Next.js 15 boilerplate for building SaaS applications. It includes 87 UI components, 8 page templates, authentication, payments, database, email, and 20 themes - everything you need to launch fast.
```

### Who is it for?
```
Fabrk is for developers who want to ship SaaS products faster. It's best suited for indie hackers, solo founders, small teams, and agencies who are tired of rebuilding the same foundations for every project.
```

### What tech stack?
```
Next.js 15 (App Router), TypeScript, Tailwind CSS, DaisyUI, Prisma ORM, PostgreSQL, NextAuth v5, Stripe, and Resend. All modern, actively maintained libraries.
```

### Is it a one-time payment?
```
Yes. $299 once, and it's yours forever. No subscriptions, no recurring fees. Use it for unlimited projects.
```

### Can I use it for client work?
```
Absolutely. The license allows unlimited projects - personal, commercial, client work, anything. The only restriction is you can't resell the boilerplate itself.
```

### What if it's not for me?
```
30-day money-back guarantee. If Fabrk isn't the right fit, email us for a full refund. No questions asked.
```
