================================================================================
TROUBLESHOOTING
================================================================================

Dev Server Won't Start:
  → Kill existing: lsof -ti:3000 | xargs kill -9
  → Then: npm run dev

Tests Time Out:
  → Use headed mode: npm run test:e2e:headed
  → Or debug mode: npm run test:e2e:debug
  → Check dev server is running

Element Not Found:
  → Run with headed browser to see page
  → Check CSS selectors with DevTools
  → Verify element is in viewport

Flaky Tests:
  → Increase timeout in playwright.config.ts
  → Use better wait conditions
  → Run multiple times to verify

More Help:
  → Read: tests/e2e/README.md (full guide)
  → Quick: tests/e2e/QUICK-START.md
  → Visit: https://playwright.dev
