# Deployment Documentation

This folder contains deployment guides for various platforms.

## Contents

- **DEPLOYMENT.md** - Primary deployment guide (Vercel recommended)
- **ALTERNATIVE-DEPLOYMENTS.md** - Deploy to Netlify, AWS, Railway, DigitalOcean

## Supported Platforms

- **Vercel** (Recommended) - One-click deployment with Next.js 15 support
- **Netlify** - Alternative serverless platform
- **AWS** - EC2, Elastic Beanstalk, or Amplify
- **Railway** - Simple container deployment
- **DigitalOcean** - App Platform or Droplets
- **Self-hosted** - Docker and systemd configurations
