# Product Hunt Launch Guide

Launch Fabrk on Product Hunt to reach 50K+ developers and generate buzz for your launch.

## Timeline

- **T-30 days**: Prepare launch materials
- **T-14 days**: Set up Product Hunt account, prepare thumbnail
- **T-7 days**: Final testing, write announcement
- **T-0**: Launch day (Tuesday-Thursday recommended)
- **T+1 to T+3**: Monitor, respond to comments, iterate

## Before Launch

### 1. Prepare Your Profile (T-14 days)

**Product Hunt Account**:
1. Create account at [producthunt.com](https://www.producthunt.com)
2. Complete your maker profile
3. Add profile photo, bio, website

**Profile Optimization**:
- Professional photo (headshot if solo founder, team photo if team)
- Compelling bio: "Building [Fabrk] to help X do Y faster"
- Links to Twitter, website, GitHub (shows credibility)

### 2. Design Thumbnail (T-14 days)

**Specifications**:
- Dimensions: 1200 x 630px
- Format: PNG/JPG
- File size: < 2MB
- Avoid text (hard to read at small sizes)

**Design Tips**:
- Use your primary color (purple)
- Show product dashboard/UI
- Include "78+ components" or "$299"
- Keep clean and minimal

**Example thumbnail elements**:
- Primary color background
- Product screenshot in center
- Large headline: "Fabrk - SaaS Boilerplate"
- Smaller subheading: "78+ components. Production-ready. $299"

### 3. Write Your Tagline (T-14 days)

**Product Hunt tagline** (60 characters max):
```
"Ship SaaS in days, not months. 77 production-ready components."
```

Alternative ideas:
- "The complete Next.js 15 SaaS boilerplate"
- "78+ components + payments + auth + docs. Launch ready."
- "3x faster to launch with production-ready code"

### 4. Create Gallery (T-7 days)

Upload 5-10 images showing:
- [ ] Dashboard overview
- [ ] Component library showcase
- [ ] Code quality (code snippet)
- [ ] Feature highlights
- [ ] Comparison chart
- [ ] Pricing page
- [ ] Team (if applicable)

**Image specs**:
- 1200 x 630px minimum
- Aspect ratio: 16:9 or square
- High quality, no compression
- Each with 1-2 sentence caption
