# Testing AI-Generated Code

Utilities and patterns for testing code generated by AI.

---

## Overview

AI generates code but often forgets to:
- Write tests
- Handle edge cases
- Validate error paths
- Check null/undefined

This guide provides testing utilities designed for AI-generated code.

## Testing Utilities

### AITest Class

Fluent API for testing AI-generated functions:

```typescript
import { AITest } from '@/lib/ai/testing';
import { userSchema } from '@/types/schemas';

describe('getUserProfile', () => {
  it('should handle all cases', async () => {
    await new AITest(getUserProfile)
      .isAsync()
      .shouldNotThrow(['user-123'])
      .shouldReturnType(userSchema, ['user-123'])
      .shouldHandleNull()
      .shouldHandleUndefined()
      .shouldHandleEmpty()
      .shouldCompleteInMs(5000, ['user-123'])
      .verify();
  });
});
```

### Available Assertions

| Method | Description |
|--------|-------------|
| `.isAsync()` | Verify function is async |
| `.shouldNotThrow(inputs)` | Function doesn't throw with inputs |
| `.shouldReturnType(schema, inputs)` | Return value matches schema |
| `.shouldHandleNull()` | Handles null input gracefully |
| `.shouldHandleUndefined()` | Handles undefined gracefully |
| `.shouldHandleEmpty()` | Handles empty string gracefully |
| `.shouldHandleErrors()` | Error handling works |
| `.shouldCompleteInMs(ms, inputs)` | Completes within time limit |
| `.verify()` | Run all assertions |

## Testing Patterns

### Async Function Test

```typescript
import { describe, it, expect, vi } from 'vitest';
import { getUser } from '@/lib/api/users';

describe('getUser', () => {
  it('returns user on success', async () => {
    // Mock fetch
    global.fetch = vi.fn().mockResolvedValue({
      ok: true,
      json: () => Promise.resolve({ id: '123', name: 'Test' }),
    });

    const result = await getUser('123');

    expect(result.success).toBe(true);
    expect(result.data?.name).toBe('Test');
  });

  it('handles not found', async () => {
    global.fetch = vi.fn().mockResolvedValue({
      ok: false,
      status: 404,
      json: () => Promise.resolve({ error: 'Not found' }),
    });

    const result = await getUser('invalid');

    expect(result.success).toBe(false);
    expect(result.error?.code).toBe('NOT_FOUND');
  });

  it('handles network error', async () => {
    global.fetch = vi.fn().mockRejectedValue(new Error('Network error'));

    const result = await getUser('123');

    expect(result.success).toBe(false);
  });
});
```

### React Component Test

```typescript
import { render, screen, fireEvent, waitFor } from '@testing-library/react';
import { UserProfile } from '@/components/user-profile';

describe('UserProfile', () => {
  it('renders loading state', () => {
    render(<UserProfile userId="123" />);

    fireEvent.click(screen.getByText('Load User'));

    expect(screen.getByText('Loading...')).toBeInTheDocument();
  });

  it('renders user data on success', async () => {
    global.fetch = vi.fn().mockResolvedValue({
      ok: true,
      json: () => Promise.resolve({ id: '123', name: 'Test User' }),
    });

    render(<UserProfile userId="123" />);
    fireEvent.click(screen.getByText('Load User'));

    await waitFor(() => {
      expect(screen.getByText('Test User')).toBeInTheDocument();
    });
  });

  it('renders error state on failure', async () => {
    global.fetch = vi.fn().mockRejectedValue(new Error('Failed'));

    render(<UserProfile userId="123" />);
    fireEvent.click(screen.getByText('Load User'));

    await waitFor(() => {
      expect(screen.getByText(/failed/i)).toBeInTheDocument();
    });
  });
});
```

### API Route Test

```typescript
import { describe, it, expect } from 'vitest';
import { POST } from '@/app/api/users/route';

describe('POST /api/users', () => {
  it('creates user with valid data', async () => {
    const request = new Request('http://localhost/api/users', {
      method: 'POST',
      body: JSON.stringify({ email: 'test@example.com', name: 'Test' }),
    });

    const response = await POST(request);
    const data = await response.json();

    expect(response.status).toBe(201);
    expect(data.data.email).toBe('test@example.com');
  });

  it('rejects invalid email', async () => {
    const request = new Request('http://localhost/api/users', {
      method: 'POST',
      body: JSON.stringify({ email: 'invalid', name: 'Test' }),
    });

    const response = await POST(request);

    expect(response.status).toBe(400);
  });

  it('requires authentication', async () => {
    // No auth header
    const request = new Request('http://localhost/api/users', {
      method: 'POST',
      body: JSON.stringify({ email: 'test@example.com', name: 'Test' }),
    });

    const response = await POST(request);

    expect(response.status).toBe(401);
  });
});
```

## Edge Case Testing

### Null/Undefined Handling

```typescript
describe('edge cases', () => {
  it('handles null input', async () => {
    const result = await processData(null);
    expect(result).toBeDefined();
    // Or expect specific error
  });

  it('handles undefined input', async () => {
    const result = await processData(undefined);
    expect(result).toBeDefined();
  });

  it('handles empty string', async () => {
    const result = await processData('');
    expect(result).toBeDefined();
  });

  it('handles empty array', async () => {
    const result = await processItems([]);
    expect(result).toEqual([]);
  });

  it('handles empty object', async () => {
    const result = await processConfig({});
    expect(result).toBeDefined();
  });
});
```

### Error Path Testing

```typescript
describe('error handling', () => {
  it('throws AppError for invalid input', async () => {
    await expect(getUser('')).rejects.toThrow('INVALID_INPUT');
  });

  it('returns error response for missing data', async () => {
    const result = await getUser('nonexistent');
    expect(result.success).toBe(false);
    expect(result.error?.code).toBe('NOT_FOUND');
  });

  it('handles database errors', async () => {
    vi.spyOn(db.user, 'findUnique').mockRejectedValue(new Error('DB error'));

    const result = await getUser('123');

    expect(result.success).toBe(false);
    expect(result.error?.code).toBe('DB_ERROR');
  });
});
```

## Type Validation Testing

### With Zod Schemas

```typescript
import { z } from 'zod';

const userSchema = z.object({
  id: z.string(),
  email: z.string().email(),
  name: z.string().min(1),
});

describe('user API response', () => {
  it('returns valid user shape', async () => {
    const response = await getUser('123');

    if (response.success && response.data) {
      const result = userSchema.safeParse(response.data);
      expect(result.success).toBe(true);
    }
  });

  it('handles missing fields gracefully', async () => {
    // Mock incomplete response
    global.fetch = vi.fn().mockResolvedValue({
      ok: true,
      json: () => Promise.resolve({ id: '123' }), // Missing email, name
    });

    const response = await getUser('123');

    // Should validate and reject incomplete data
    expect(response.success).toBe(false);
  });
});
```

## Performance Testing

```typescript
describe('performance', () => {
  it('completes within timeout', async () => {
    const start = Date.now();
    await getUser('123');
    const duration = Date.now() - start;

    expect(duration).toBeLessThan(5000);
  });

  it('handles concurrent requests', async () => {
    const requests = Array(10).fill(null).map(() => getUser('123'));
    const results = await Promise.all(requests);

    results.forEach(result => {
      expect(result.success).toBe(true);
    });
  });
});
```

## Mocking

### Mock Fetch

```typescript
import { vi } from 'vitest';

beforeEach(() => {
  global.fetch = vi.fn();
});

afterEach(() => {
  vi.restoreAllMocks();
});

it('calls API correctly', async () => {
  global.fetch = vi.fn().mockResolvedValue({
    ok: true,
    json: () => Promise.resolve({ data: 'test' }),
  });

  await myFunction();

  expect(fetch).toHaveBeenCalledWith(
    expect.stringContaining('/api/'),
    expect.objectContaining({ method: 'GET' })
  );
});
```

### Mock Database

```typescript
import { vi } from 'vitest';
import { db } from '@/lib/db';

vi.mock('@/lib/db', () => ({
  db: {
    user: {
      findUnique: vi.fn(),
      create: vi.fn(),
      update: vi.fn(),
    },
  },
}));

it('creates user in database', async () => {
  vi.mocked(db.user.create).mockResolvedValue({
    id: '123',
    email: 'test@example.com',
    name: 'Test',
  });

  const result = await createUser({ email: 'test@example.com', name: 'Test' });

  expect(db.user.create).toHaveBeenCalledWith({
    data: { email: 'test@example.com', name: 'Test' },
  });
});
```

## Coverage Requirements

Recommended minimums:

| Metric | Target |
|--------|--------|
| Statements | 80% |
| Branches | 75% |
| Functions | 80% |
| Lines | 80% |

Configure in `vitest.config.ts`:

```typescript
export default defineConfig({
  test: {
    coverage: {
      provider: 'v8',
      thresholds: {
        statements: 80,
        branches: 75,
        functions: 80,
        lines: 80,
      },
    },
  },
});
```

3. Test null/undefined handling

5. Verify return types match schema
6. Check performance within limits

await new AITest(myFunction)
